Message To display your pool under 'Mined By' please edit your S-NOMP hex string in: 

/s-nomp/config.json

Find:


    "defaultPoolConfigs": {
        ...
	"poolHex": "457175694875622068747470733A2F2F627A656467652E657175696875622E70726F"
    },

HEX encode a string containing the name of your pool and the URL and replace the above with your own HEX string. The encoded string should be of the following format:

YourPoolName https://your.pool.url
Then submit your HEX string to #pools on the BZedge Discord server.
